## Project Name
XYZ Meeting Management System

## Description
This project is a meeting management system for employees. The system allows employees to create meetings, view meeting schedules, manage meeting status, and check meeting details.

## Main Features
- View meeting dashboard
- Create new meeting
- View meeting details
- Filter meetings by date, week, month, and status
- Manage employee profile
